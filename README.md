# 天機算命 - AI 八字大師

這是基於 Google Gemini AI 的八字算命應用程式，結合傳統子平八字學理與現代 AI 技術，為您提供深入的命理分析。

## 特色功能

- **正宗八字排盤**：精確換算真太陽時與節氣 (支援 1900-2100 年)，並支援三柱推算。
- **AI 大師論命**：
    - **八字論命**：分析格局、用神。**[NEW]** 新增「五行能量視覺化」圖表，強弱一目瞭然。
    - **流年運勢**：針對 2025 (乙巳)、2026 (丙午) 等流年進行吉凶預測，包含太歲互動與流年神煞。
    - **古籍考據**：引用《三命通會》、《滴天髓》等古文進行深度學術探討。
    - **雙人合盤**：支援「合婚/合盤」模式，分析兩人契合度與相處建議。
- **貼心功能**：
    - **[NEW] 神煞列表**：直接顯示每柱神煞（如天乙貴人、咸池），不再只有文字描述。
    - **[NEW] 每日靈籤**：每日一抽，獲取今日運勢指引與開運顏色/數字。
    - **[NEW] 互動式追問**：AI 自動生成 3 個最迫切的改運建議問題，一鍵追問。
    - **[NEW] PWA 支援**：可安裝至手機桌面，離線也能查看部分介面。
    - **隱私優先**：承諾「無伺服器儲存」，您的命盤僅在瀏覽器與 Google API 間傳輸。
- **雙重解讀風格**：
    - **白話解讀**：淺顯易懂，適合一般大眾。
    - **古籍專業**：模仿徐樂吾大師風格，適合命理研究者。
- **在地化設計**：全繁體中文介面，符合台灣用語習慣。

## 技術架構

- **前端**：React 18 + TypeScript + Vite
- **樣式**：Tailwind CSS
- **AI 模型**：Google Gemini 2.0 Flash (via `@google/genai`)
- **部署**：GitHub Pages

## 本地開發

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 設定 API Key：
   在根目錄新增 `.env` 檔案，並填入您的 Gemini API Key：
   ```
   VITE_API_KEY=your_api_key_here
   ```

3. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

4. 建置生產版本：
   ```bash
   npm run build
   ```
